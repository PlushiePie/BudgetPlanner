 Планировщик трат — Управление личным бюджетом

Приложение для учёта личных финансов, контроля бюджета и аналитики трат. Написано на Kotlin + Swing.

 Возможности

 Основные функции
- Добавление трат (сумма, категория, комментарий, дата)
- Просмотр всех трат в одном списке
- Удаление ненужных трат (правый клик мыши)
- Отметка выполненных трат

 Управление бюджетом
- Ручная установка общего бюджета на месяц
- Редактирование бюджета для каждой категории отдельно
- Визуальные прогресс-бары с цветовой индикацией:
  - Зелёный - до 80% бюджета
  - Оранжевый - 80-100%
  - Красный - превышение

 Аналитика и прогнозы
- Средние траты - в день, неделю, месяц
- Самая затратная категория - сумма и процент от всех трат
- Прогноз на конец месяца - сколько останется денег
- Дополнительная статистика - всего трат, средний чек, дней с тратами

 Другое
- Автосохранение - все данные сохраняются при выходе
- Хранение в файле - при следующем запуске всё на месте

 Навигация по меню

| Пункт меню | Что делает |
|------------|------------|
| Добавить трату | Открывает окно для новой траты |
| Просмотреть все траты | Показывает полный список трат |
| Редактировать бюджеты | Меняет бюджет для каждой категории |
| Установить бюджет на месяц | Задаёт общий бюджет (категории пересчитаются сами) |
| Прогнозы и аналитика | Показывает статистику и прогнозы |

 Сохранение данных

Все данные автоматически сохраняются в файл budget_data.ser в папке с программой. При повторном запуске бюджет и траты загружаются автоматически.

 Технологии

- Kotlin - язык программирования
- Swing - графический интерфейс (встроен в Java/Kotlin)
- Сериализация - сохранение данных
